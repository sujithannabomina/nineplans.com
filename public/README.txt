NinePlans logo assets (drop-in for /public)

Recommended usage:
- public/logo.png               -> header / navbar logo (mark-only, padded square)
- public/logo-mark.png          -> mark-only transparent (use anywhere)
- public/logo-full.png          -> full logo (with text) transparent
- public/favicon.ico            -> classic favicon
- public/favicon-16x16.png      -> favicon
- public/favicon-32x32.png      -> favicon
- public/apple-touch-icon.png   -> iOS home screen
- public/android-chrome-192x192.png
- public/android-chrome-512x512.png
- public/mstile-150x150.png     -> Windows tile
- public/og-image.png           -> social share image (1200x630)

If your project already has files with the same names in /public,
REPLACE them with these files.
