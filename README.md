# NinePlans (React + Firebase)


Anonymous posting, search, report button, moderation queue, and community moderators.


## Quick start
1. `npm i`
2. Add your Firebase config in `src/firebase.js`
3. Firebase Console → Firestore → **Rules**: paste `firestore.rules`
4. Firebase Console → Authentication → enable **Anonymous** and **Google**
5. Start: `npm run dev`
